Blockchain Program

Powered by Go (Golang)

With the massive rise in interest of cryptocurrency and blockchain technologies in 2020, I thought it would be cool to program a simple blockchain structure that contains its own blocks and validates each block through proof-of-work (PoW) mining.

I chose Go as the blockchain's technology to take advantage of its in-built concurrency features. PoW mining is a power-intensive activity that requires calculating several hash values per second. Popular cryptocurrencies like Bitcoin and Ethereum (for now) use PoW to validate blocks containing sets of transactions. In order to decrease hashing time to validate a block, utilizing multi-core processors would be ideal. Go easily allows a developer to instantiate multiple user threads through goroutines and go channels.

By using Go concurrency features to validate blocks for my blockchain, I increased the efficiency of proof-of-work validation for my blockchain. I gained a better understanding and appreciation for blockchain technologies, and I believe that blockchain adoption will become more prevalent in the future.
