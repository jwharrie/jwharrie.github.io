Earnings Calculator

Powered by React, Node.js, HTML & CSS

After graduating with my CS degree and getting inspired by the GameStop short squeeze of 2021, I became more interested in personal financing and investing. One of my challenges was determining the best savings accounts offered to Canadians that range from non-registered accounts to TFSAs. I needed to find a way to calculate compound interest earnings for each account so that I can improve my financial planning. The solution for me was to develop a React earnings calculator.

The earnings calculator is a React web app that can calculate account balances and interest earnings for one or more bank accounts. Each bank account can set its own initial balance, annual percentage rate (APR), monthly contribution and time period of accumulation in months. Users can add or delete multiple accounts and see their total balance and earnings among all accounts. You can even set an investment goal for each account to calculate the number of months or years required to reach it.

React provided me the ideal technology for the calculator. React components increased modularity of my code while re-rendering after each state change made my app more response and intuitive. Using JSX for each component simplified the creation of html elements and therefore made development easier. I can understand why developers opt for React as a front-end technology.
