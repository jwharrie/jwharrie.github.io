Twitter Spell Correction Program

Powered by Python

One of the more interesting courses I took at Simon Fraser University was about information retrieval (IR) and web search. Some of the topics taught included webscraping and text processing, both handy for my Twitter spell correction program.

The purpose of the program is to retrieve the 1000 most recent tweets, identify mispelled words and suggest spelling corrections for each word. I was required to gain access and understand Twitter's API. Afterwards I needed a way to seamlessly utilize the API, retrieve the tweets, process each tweet for words and analyze the word collection for spelling mistakes.

The solution was to write a Python program that took advantage of open-source, readily available and essential modules. The module tweepi allowed me to easily access Twitter through the Twitter API in Python and write a stream of tweets into a .txt file. Python's regex module, re, removed URLs and punctuation marks found in Tweets. nltk, a natural language processing module, supplied a list of English stopwords, insignificant words that occur very frequently ('a', 'the', 'of', etc.). I used the list to filter out stopwords found in the Tweets to make my word collection only filled with significant and interesting words. pandas dataframes provided great versatility and convenience in applying word processing functions to my collection of Tweets.

After creating my Tweet word collection I began searching for mispelled words and their suggested word corrections. I created a Python dictionary containing each word's frequency, then I made a series of assumptions about what a mispelled word would most likely be to avoid falsely selecting a correcly-spelled word. With these assumptions and the dictionary I successfully detected mispelled words and generated a list of spelling suggestions for each one.
