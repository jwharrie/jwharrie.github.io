Mine Seeker Mobile Game

Powered by Java & Android

Developing video games are a great way to learn mobile development. Android smartphones hold a much larger market share than iPhones and Android development is more accessible. For these reasons, I chose to develop a game for Android devices.

The game I developed, Hack the Pentagon, is known as a mine seeker game. The goal takes place on a grid layout and the goal is to seek all mines (or in this case, classified Pentagon files!) with the fewest guesses possible. Selecting a spot with a mine reveals it while selecting one without a mine shows a number that counts the amount of hidden mines in the same row and column of the spot. These numbers update as more mines are uncovered.

Android development was fun for me. I used the Android Studio IDE which allowed me to easily create view layouts paired with a controller file that can access model data. I used the MVC design pattern to organize the app as well as the Singleton pattern to share the same data throughout the app. Java's class-centered nature came in handy for the object-oriented programming (OOP) approach used to organize and develop the app. Overall, I became more familiar with mobile development and increased my OOP abilities from developing the game.
