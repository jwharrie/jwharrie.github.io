Video Rental Cloud Database App

Powered by Python, SQL & Azure

Cloud computing is becoming more prevalent in our lives and one of the most well-known cloud services offered is Microsoft Azure. Taking advantage of their free trial I created an Azure SQL server to become more familiar with cloud services and reinforce my understanding of relational databases.

The project simulates a video rental database that contains 4 tables for movies, customers, rental plans and rental records. Upon initializing the database, records from .txt files are loaded into the 4 tables. Afterwards, users can conduct actions, like renting a movie or checking who is renting a specific movie, through a web API.

Using Python for the project provided 2 key technologies. One was pyodbc, an open-source module that allowed me to easily connect to the Azure server, execute SQL commands and retrieve query results. The other was Flask, a micro web framework that allowed me to implement a web api that users can access.

Completing this project was a great learning experience for me. I gained greater familiarity with Microsoft Azure and Python database tools. I also now know how to implement a web API with Python and increased my SQL skills.
